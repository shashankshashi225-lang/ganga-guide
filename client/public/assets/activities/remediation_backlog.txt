## PHASE 1: CRITICAL FIXES (Week 1-2)

### 1. CRITICAL-001: Set up Google Search Console & Verify Indexing
- **Severity**: CRITICAL | **Effort**: SMALL | **Owner**: Dev/SEO
- **Steps**:
  1. Verify domain ownership in Google Search Console (via DNS/HTML file)
  2. Submit sitemap.xml at https://gangaguide.com/sitemap.xml
  3. Request indexing for homepage and top 5 landing pages
  4. Monitor "Coverage" report for crawl errors, excluded URLs
  5. Check "URL Inspection" tool for each key page
- **Evidence to collect**: Screenshot of GSC dashboard showing indexed pages, crawl stats

### 2. CRITICAL-002: Mobile Responsiveness Audit & Fix
- **Severity**: CRITICAL | **Effort**: MEDIUM | **Owner**: Dev
- **Steps**:
  1. Test homepage on Google Mobile-Friendly Test: https://search.google.com/test/mobile-friendly
  2. Check viewport meta tag in HEAD: `<meta name="viewport" content="width=device-width, initial-scale=1">`
  3. Audit all pages for responsive CSS (no fixed widths, use flexbox/grid)
  4. Test touch targets: all clickable elements must be ≥48px
  5. Fix any layout shifts (CLS issues) on mobile
  6. Test on real devices (iPhone, Android)
- **Evidence to collect**: Mobile-Friendly Test pass/fail, screenshot comparison desktop vs mobile

### 3. CRITICAL-003: Performance Optimization (Core Web Vitals)
- **Severity**: CRITICAL | **Effort**: LARGE | **Owner**: Dev
- **Steps**:
  1. Run Google PageSpeed Insights for desktop and mobile: https://pagespeed.web.dev
  2. Identify Largest Contentful Paint (LCP) bottleneck (target: <2.5s)
  3. Enable GZIP compression on server
  4. Minify CSS and JavaScript files
  5. Enable lazy loading for below-fold images: `loading="lazy"`
  6. Implement image optimization (WebP with fallback)
  7. Minimize render-blocking resources (CSS/JS)
  8. Enable browser caching (Cache-Control headers)
  9. Consider CDN for static assets (Cloudflare Free tier)
  10. Remove unused CSS/JavaScript
- **Evidence to collect**: PageSpeed Insights scores (before/after), Lighthouse report PDF

### 4. CRITICAL-004: SSL/TLS & Security Headers
- **Severity**: CRITICAL | **Effort**: MEDIUM | **Owner**: Dev/Hosting
- **Steps**:
  1. Verify HTTPS on all pages (use: curl -I https://gangaguide.com)
  2. Add 301 redirect from HTTP to HTTPS in .htaccess or nginx.conf:
     ```
     RewriteEngine On
     RewriteCond %{HTTPS} off
     RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
     ```
  3. Add HSTS header (in .htaccess):
     ```
     Header set Strict-Transport-Security "max-age=31536000; includeSubDomains"
     ```
  4. Add security headers (in .htaccess):
     ```
     Header set X-Content-Type-Options "nosniff"
     Header set X-Frame-Options "SAMEORIGIN"
     Header set X-XSS-Protection "1; mode=block"
     Header set Referrer-Policy "strict-origin-when-cross-origin"
     Header set Content-Security-Policy "default-src 'self'; script-src 'self' https://www.googletagmanager.com; style-src 'self' 'unsafe-inline'"
     ```
  5. Test with Security Headers checker: https://securityheaders.com
- **Evidence to collect**: SSL Labs A+ rating, Security Headers report, curl headers output

### 6. HIGH-002: Create & Submit Sitemap & Robots.txt
- **Severity**: HIGH | **Effort**: MEDIUM | **Owner**: Dev/SEO
- **Steps**:
  1. Create sitemap.xml with all URLs (max 50K URLs per file):
     ```xml
     <?xml version="1.0" encoding="UTF-8"?>
     <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
       <url>
         <loc>https://gangaguide.com</loc>
         <lastmod>2025-12-06</lastmod>
         <changefreq>weekly</changefreq>
         <priority>1.0</priority>
       </url>
       <url>
         <loc>https://gangaguide.com/tours</loc>
         <lastmod>2025-12-06</lastmod>
         <changefreq>daily</changefreq>
         <priority>0.9</priority>
       </url>
     </urlset>
     ```
  2. Create robots.txt at /robots.txt:
     ```
     User-agent: *
     Allow: /
     Disallow: /admin/
     Disallow: /private/
     Disallow: /*.pdf$
     Sitemap: https://gangaguide.com/sitemap.xml
     Crawl-delay: 1
     ```
  3. Submit sitemap to GSC
  4. Verify no important pages are blocked
- **Evidence to collect**: Sitemap XML file, robots.txt screenshot
